Basic principles of Web Accessibility (Written in my own, awesome text)

Web accessibility, to me, means both how easy it is for users to understand and navigate a site in question. These can fluctuate based on website quality, but generally speaking a site should have easy-to-follow navigation links placed in obvious but non-obstructive places, it should have complimentary colours and styles and have a consistent styling. This is linked to and aids in understanding content on a page as having readily avaliable information (such as text and/or videos) in easy-to-find places makes content easier to digest and possibly more fun to digest.

Meaning of HTML validation (Composed of my own original, funky words)

In basic terms, HTML validation is a way to check if code is compiling in a way that doesn't run in to errors and points out any errors/mistakes in code for the user to see and fix. This can be done with a pre-existing tool that will do something such as highlighting errors in code to give a visual indication of an error.
